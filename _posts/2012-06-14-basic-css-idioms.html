Simpler columns

Say you want to have a container with two equally-sized columns in it: 
<div id="wrapper">
    <div class="column">Column 1</div>
    <div class="column">Column 2</div>
</div>
The standard way to do this is to apply a float to each column and give them a width of 50%. However, what if your columns have borders? What if you want to pad them? You'll end up with something like this: 
.column
    border: 1px solid #ccc
    float: left
    padding: 10px 2%
    width: 45%
But you end up with an imperfect box model. In addition, if you try to get the height of the #wrapper, you'll notice that the height is now 0. Let's fix both of these: 
#wrapper
    overflow: hidden

.column
    border: 1px solid #ccc
    box-sizing: "border-box"
    float: left
    padding: 10px
    width: 50%
The overflow: hidden bit will fix our box height. This is the typical "clearfix" that you may have seen. Some people like to put a div with clear: both attributes, but that's adding unnecessary markup for most cases. The second trick here is the box-sizing: "border-box" declaration. This let's us compute the width including the borders and padding (but not margins). Now 50% really means half.

Offset positioning

The idiom here is a pretty good one to remember: an absolute-positioned element inside a relative-positioned element can be positioned relative to the container. Here's the example: 
<div id="outside" style="height: 300px">
    <div id="inside">Move me around!</div>
</div>
Right now, the "inside" element is positioned at the top of the "outside" element. To get it to stick to the bottom, you could simply do this: 
#outside
    position: relative

#inside
    bottom: 0
    position: absolute
The values for position become a little confusing in this situation, but it works beautifully. Another common case I run into is when using font-based icons like Pictos. To position the icon exactly where I want it, I can use relative/absolute positioning. The HTML: 
<p>My text</p>
A bit of SASS: 
p
    padding-left: 20px
    position: relative
    &:before
        content: "i"
        font-size: 18px
        left: -5px
        position: absolute
        top: -4px
